Introduction to Contented5

Contented5 is the fifth open source template from Contented Designs. It's valid CSS, valid XHTML 1.0 strict, and it's been tested for consistent appearance on:

  • Internet Explorer 6.0.2900.2180
  • Firefox 1.5.0.3
  • Opera 8.53

Current version: Contented5 Version 1.01 (May 17, 2006)

Using this Design

Contented5 is an open source template. You're free to modify it and use it for any purpose without cost or obligation. We prefer that you leave the link to our website in the footer but it's not required.

If you have comments or questions, please contact us at Contented Designs. Thanks!
